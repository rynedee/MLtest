Welcome to The Watts Times, a Modern Labor Application Project

This page is a result of my first forrays into web development. It's basic design belies the sweat and tears that went into creating it but it also taught some valuable lessons. For instance, while I don't completely understand flexbox I do realize how much easier it makes laying out a webpage.

While I learned a lot it quickly became apparant that I was in over my head. Creating web pages in 2019 is vastly different than creating pages in 1998, the last time I seriously delved into web development. It feels more modular, less linear. At any rate it was readily apparent that I needed to do more than 'review' HTML, to that end I've begun learning JavaScript and Python in addition to reaquinting myself with HTML.

My background in all this is sporadic. My introduction into computer science extends all the way back to the mid-80s where my dad taught some really basic commands that allowed me to look at a program's code on a TRS-80. I was able to use that knowledge to debug some Apple II software a few years later.

It wasn't until my sophomore year in college that I returned to learning code when a professor reccommed I take a programming course, Fortran if I remember correctly. I was confused from the start, it didn't help that I added the course a month into the semester and I was already taking Organic Chemistry, and ended up failing the class. A few years later the Communications department offered a web marketing class that tasked us with building a web presense for a local business.

Since then I haven't done much in the realm of software development. I did dabble into HTML for a project I started while living in Uzbekistan that would detail an NBA season from the perspective of an expat but it never progressed past a beginning tempormental prototype.

Anywho, hopefully you enjoy my little slice of the web. Be sure to try my Hangman game, just click the green play button get started. Be sure to drag the bar to the top to maximize your window.
